# Janda-JajanDenganDamai
Digitalisasi sistem "jajan" di sekolah dengan mengintegrasikannya pada sebuah web app interaktif untuk meningkatkan efisiensi

Sebuah sistem web "jajan" yang dapat digunakan oleh siswa. Mengadaptasikan sistem yang sama seperti Gojek namun lebih berfokus pada kasus jajan anak sekolahan. Minimarket serta booth store yang ada di sekolah bisa membuka "warung digital" mereka dalam platform Janda. Siswa kemudian bisa memilih jajan yang hendak mereka beli, memilih metode pembayaran, serta metode pick-up. Kemudian, Janda juga menyediakan mekanisme membership. Dengan membership, siswa dapat memperoleh diskon untuk ketentuan tertentu.
